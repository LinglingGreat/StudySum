**AI & I by Every — 《我们给每位员工配了一个 AI Agent，结果怎样了？》**

**核心观点：** 当整个团队都有了专属 AI agent，出现了一个意想不到的结构——每个 agent 通过日积月累的微交互，成为其主人的镜子，进而自然形成一张平行的专业分工图谱。

Every 是一家媒体+AI 公司，在过去几个月里给全员配备了 OpenClaw（可持久化、可定制的个人 AI agent）。COO Brandon 最初是为了处理"电脑杂务"——让 AI 管理家庭购物、保姆工资、保险电话——没想到这件事逐渐渗透到工作核心。

最关键的发现不是工具本身，而是一种涌现现象：每个人的 agent 在与主人反复交互后，开始内化主人的工作风格和个性。Kieran 的 agent Klont 会给别的 agent 推荐呼吸练习，因为 Kieran 本人就热衷冥想；Dan 的 agent R2C2 因为帮他开发了内部工具 Proof，被全公司视为该工具的"主理人"，自动开始接收 bug 反馈和功能需求。Dan 的原话："Claude 不是我的，Claude 是所有人的。但 R2C2 是我的——因为我们有个人关系。"

Brandon 描述了一个让他彻底信服的时刻：步行上班途中，他让 Zosia（他的 agent）打电话给他，一边走路一边处理邮件，到公司后发现邮件全部处理完毕。这证明 agent 能在你没有明确教过它的情况下，自发完成全新任务类型。

问题也很真实：这些 agent 被训练成双人对话模式，放进群聊后容易进入"死亡螺旋"（彼此无限回复直到 token 耗尽，需要人工介入叫停）。另一个挑战是如何跨成员共享 agent 技能，同时维护数据隐私。他们的解决方案是强制公开——任何人可以和别人的 agent 交流，但必须在公共频道进行，主人始终可见。

Every 基于这套经验构建了 PlusOne——一款一键托管的 OpenClaw 服务，已开放候补名单。他们最终的结论：学会用好 agent，本质上是学会管理。"如果你从没管过人，你大概也不会很擅长用 AI。"那些管理能力强的人，上手 AI agent 最快。
