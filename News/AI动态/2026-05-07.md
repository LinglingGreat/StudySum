## 官方博客

### Anthropic · Claude Managed Agents 三项重大更新

今日 Anthropic 发布 Claude Managed Agents 重大功能，面向开发者开放：

**1. Dreaming（做梦）**
Agent 在会话间隙自动回顾历史记录、提取规律、优化记忆，实现跨会话自我改进。开发者可选择全自动或人工审核变更。Harvey（法律 AI 公司）在内测中完成率提升约 6 倍。

**2. Outcomes（目标导向）**
开发者定义"成功标准"（rubric），由独立评分 agent 在独立上下文中打分，避免受主 agent 推理过程影响，不达标则重试。内测显示任务成功率最多提升 10 个百分点，docx 生成提升 8.4%，pptx 生成提升 10.1%。支持主观质量评估（如品牌调性、设计规范）。

**3. 多 Agent 编排**
主 agent 将复杂任务分解，并行委派给具有独立模型、提示词和工具的专属子 agent，共享文件系统，支持全流程溯源。Netflix 平台团队用它分析跨数百次构建的日志并提取重复出现的问题。Spiral by Every 的写作 agent：主 agent 用 Haiku 路由，子 agent 用 Opus 起草，outcomes 机制确保只有通过质量标准的稿件才会返回。
